Group project for Computer Organization CSCI2500 - Pipelined MIPS Simulator - Fall 2018

Names:

Run `> python3 p1.py [mode] [input]`

`[mode]` can be "N" for no-forwarding mode or "F" for forwarding mode.

`[input]` is a MIPS input file.
